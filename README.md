## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/heart-disease-predictor.git
cd heart-disease-predictor
```

### 2. Set up a virtual environment
```bash
python -m venv venv
source venv/bin/activate        # macOS / Linux
venv\Scripts\activate           # Windows
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Train the model
```bash
python train.py
```
This generates `final_model.pkl` and saves performance plots to `plots/`.

### 5. Launch the app
```bash
streamlit run app.py
```
Open your browser at **http://localhost:8501**

## 📊 Dataset

The model is trained on the **UCI Heart Disease dataset** (`heart.csv`):

| Feature | Description |
|---|---|
| `age` | Age in years |
| `sex` | 0 = Female, 1 = Male |
| `cp` | Chest pain type (0–3) |
| `trestbps` | Resting blood pressure (mm Hg) |
| `chol` | Serum cholesterol (mg/dl) |
| `fbs` | Fasting blood sugar > 120 mg/dl |
| `restecg` | Resting ECG results (0–2) |
| `thalach` | Maximum heart rate achieved |
| `exang` | Exercise-induced angina |
| `oldpeak` | ST depression (exercise vs rest) |
| `slope` | Slope of peak exercise ST segment |
| `ca` | Number of major vessels (0–4) |
| `thal` | Thalassemia type (1–3) |
| `target` | **0 = Healthy, 1 = Heart Disease** |

## 🤖 Model Details

- **Algorithm:** XGBoost Classifier
- **Preprocessing:** Standard Scaling
- **Validation:** 5-Fold Stratified Cross-Validation
- **Key metrics (held-out test set):**
  - Accuracy: ~98%
  - ROC-AUC: ~0.99
  - F1-Score: ~0.98

## ☁️ Deploy to Streamlit Cloud

1. Push the repo to GitHub (make sure `heart.csv` is included)
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub account and select the repo
4. Set **Main file path** to `app.py`
5. Add a startup command in `.streamlit/config.toml` to run `train.py` first (see below)

**`.streamlit/config.toml`** (optional):
```toml
[server]
headless = true
port = 8501
```

> Note: `final_model.pkl` must exist before `app.py` runs. On Streamlit Cloud, add a `setup.sh` or handle training inside the app on first launch.
